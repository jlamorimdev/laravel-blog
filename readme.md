## Requisitos

PHP >= 7.1.3

OpenSSL PHP Extension

PDO PHP Extension

Mbstring PHP Extension

Tokenizer PHP Extension

XML PHP Extension

Ctype PHP Extension

JSON PHP Extension

BCMath PHP Extension

Node >= 8

NPM >= 6

### Orientações

##### Faça um clone deste repositório

git clone https://github.com/jlamorimdev/laravel-blog.git

Bash
```bash
$ cd laravel-blog
$ composer install
$ cp .env.example .env
$ php artisan key:generate
$ touch database/database.sqlite
$ php artisan migrate --seed
$ npm install
$ npm run dev
$ php artisan serve 
```
